# PDF Editor

A simple, browser-based PDF editor for viewing, annotating, and editing PDF documents.

## Features

- 📄 **View PDFs** - Render and navigate through PDF pages
- ✏️ **Annotate** - Add text, highlights, and drawings
- 🔲 **Shapes** - Draw rectangles, circles, and arrows
- 💾 **Export** - Download edited PDF

## Tech Stack

- React 18 + TypeScript
- Vite
- PDF.js (rendering)
- pdf-lib (editing)
- Tailwind CSS

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Build

```bash
npm run build
```
